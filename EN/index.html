Hamilton Circuit

A program made in C# that searches connections between circles from an image full of black circles and a white background. It avoids making a connection where there is an obstacle (another circle or a line). This program finds the closest circles too, indicating them to the image and making a Hamiltoninan circuit in case there is one, and in case there is, an animation is made along the path to complete the circuit.

Prim y Kruskal

A program made in C# that searches connections between circles from an image full of black circles and a white background. It avoids making a connection where there is an obstacle (another circle or a line). Then the program runs Prim and Kruskal algorithms to the graph obtained, showing a path at the respective algorithm as an animation with a little particle. These particles will decrease in size when they take a branch from the overlapping tree.
